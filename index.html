Quetzal

Quetzal is an experimental translation of concepts from the QuickUI framework to HTML custom elements. This relies on Google's Polymer project to support aspects of custom elements which are not yet supported in all browsers. The Quetzal source code is available on GitHub.

A small handful of initial, experimental custom elements created in Quetzal appear below. These are translations of some components from the QuickUI Catalog. For the time being, use Google Chrome to view this page.

color-swatch Shows a small square of a CSS color. document-icon-button Demonstrates how to pre-fill a base class property, in this case an icon. Open icon-button A button with an optional icon. Reload labeled-color-swatch Interprets its content as a CSS color and adds a corresponding swatch of color. LightSeaGreen lorem-pixel Selects a random photo from lorempixel.com. mapped-list Maps its children to a corresponding set of instances of a desired element class. Red Blue Green markup-tag Formats its contents as an HTML tag. Here's a reference to the script tag. quetzal-button Shows its content inside a simple button. Note: Buttons currently have extra padding because — for some as yet undetermined reason — their content area does not appear to respond to clicks. They appear pressed only when clicked within their padding. Button quetzal-element A plain Quetzal element. This currently behaves like a div. repeat-list Creates a certain number of instances of another element class.
