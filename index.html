Quetzal

Quetzal is an experimental translation of concepts from the QuickUI framework to HTML custom elements. This relies on Google's Polymer project to support aspects of custom elements which are not yet supported in all browsers. The Quetzal source code is available on GitHub.

A small handful of initial, experimental custom elements created in Quetzal appear below. These are translations of some components from the QuickUI Catalog. For the time being, use Google Chrome to view this page.

auto-size-text-box A text area that expands to contain its text. icon-button A button with an optional icon.
